# Message Variations - Practical Examples

## Basic Examples

### Example 1: Simple Greeting

```
Olá|Oi|E aí, tudo bem?
```

**Possible outputs:**
- "Olá, tudo bem?"
- "Oi, tudo bem?"
- "E aí, tudo bem?"

**Total combinations:** 3

---

### Example 2: Greeting with Name Variable

```
Olá|Oi|E aí {{nome}}, tudo bem?|como vai?|beleza?
```

**With variables:** `{ nome: "João" }`

**Possible outputs:**
- "Olá João, tudo bem?"
- "Oi João, como vai?"
- "E aí João, beleza?"

**Total combinations:** 9 (3 × 3)

## Tips for Creating Good Variations

### 1. Keep the Same Meaning

✅ Good:
```
Olá|Oi|E aí
```

❌ Bad:
```
Olá|Tchau|Talvez
```

### 2. Maintain Consistent Tone

✅ Good (Formal):
```
Bom dia|Olá|Prezado
```

✅ Good (Informal):
```
Oi|E aí|Fala
```

❌ Bad (Mixed):
```
Prezado|E aí|Fala
```

### 3. Use Natural Language

✅ Good:
```
Gostaria de|Queria|Posso te mostrar
```

❌ Bad:
```
Gostaria de|Desejo|Almejo te mostrar
```

### 4. Test Your Variations

Always use the preview feature to see how your variations look in practice.

### 5. Don't Overdo It

✅ Good (3-5 variations):
```
Olá|Oi|E aí|Fala|Opa
```

❌ Bad (too many):
```
Olá|Oi|E aí|Fala|Opa|Salve|Coé|Beleza|Tudo bem|Como vai
```

## Advanced Patterns

### Pattern 1: Conditional Variations

```
{{#if vip}}
Olá {{nome}}, cliente VIP!|Prezado {{nome}}, nosso cliente especial!
{{else}}
Olá|Oi {{nome}}!
{{/if}}
```

### Pattern 2: Time-based Variations

```
{{#if morning}}
Bom dia|Olá {{nome}}, bom dia!
{{else}}
Boa tarde|Olá {{nome}}, boa tarde!
{{/if}}
```

### Pattern 3: Segmented Variations

```
{{#if segment == 'premium'}}
Olá {{nome}}, temos uma oferta exclusiva|especial para você!
{{else}}
Olá|Oi {{nome}}, confira nossa promoção!
{{/if}}
```

## Testing Checklist

Before sending your message with variations:

- [ ] Preview shows different combinations
- [ ] All variations have the same meaning
- [ ] Tone is consistent across variations
- [ ] Variables are working correctly
- [ ] No syntax errors
- [ ] At least 2 variations per block
- [ ] Not more than 10 variations per block
- [ ] Message makes sense in all combinations
